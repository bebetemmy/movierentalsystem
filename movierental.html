Python Movie Rental System Learning

Welcome to the 4-week course to help you build a movie rental system using Python. Follow each week step-by-step with theory, activities, and coding exercises.

Week 1: Python Basics & User Interaction

Key Concepts

  • Variables (e.g., movie_a_rented = False)
  • Data types (strings, booleans)
  • input() and print() functions
  • String methods (e.g., .strip())

Exercise 1

Write a script that asks for the user's name and prints a greeting.

name = input("Enter your name: ") print(f"Hello {name}!")

Exercise 2

Ask for age and display a movie eligibility message.

age = int(input("Enter your age: ")) if age >= 18: print("You can rent any movie.") else: print("Some movies may be restricted.")

Week 2: Control Flow and Menu Systems

Key Concepts

  • while loops
  • if / elif / else
  • Menu-driven logic
  • String comparison

Exercise

Write a menu-driven program that says hello or exits based on user choice.

while True: print("1. Say Hello\n2. Exit") choice = input("Choose an option: ") if choice == '1': print("Hello there!") elif choice == '2': break else: print("Invalid option.")

Week 3: Data Structures & Modularity

Key Concepts

  • Dictionaries, lists
  • Functions and modular code
  • DRY (Don't Repeat Yourself) principle

Exercise

Display available movies using a function.

movies = {"Movie A": False, "Movie B": False} def show_available(): for movie, rented in movies.items(): if not rented: print(movie) show_available()
